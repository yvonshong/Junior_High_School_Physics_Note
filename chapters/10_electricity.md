## 十、 电学、电路与定量实验 (Electricity)

### （一）部分电路基础
1. 电荷吸引轻小物体。静电通过摩擦起电发生转移。正电荷由丝绸摩擦玻璃棒产生；负电荷由毛皮摩擦橡胶棒产生。
2. 验电器：利用同种电荷相互排斥的原理来检验物体是否带电。
3. 元电荷 $$e = 1.6 \times 10^{-19}\text{ C}$$。电量单位库仑，常用单位毫安时 ($$\text{mAh}$$)，$$1\text{ mAh} = 3.6\text{ C}$$。$$1\text{ C} = 6.25 \times 10^{18}e$$。电流做功公式 $$W = UIt = UQ = Pt$$。
4. **电阻定律决定式**：$$R = \rho\frac{L}{S}$$。金属导体电阻率随温度升高而增大（液体除外）。**欧姆定律计算式**：$$I = \frac{U}{R}$$ ；闭合电路欧姆定律 $$I = \frac{E}{R+r}$$。
5. 电功与电功率比例分配：
    * 串联分压：$$\frac{U_1}{U_2} = \frac{R_1}{R_2}$$ ，且总电阻 $$R = R_1 + R_2$$。串联功率分配：$$\frac{P_1}{P_2} = \frac{R_1}{R_2}$$。
    * 并联分流：$$\frac{I_1}{I_2} = \frac{R_2}{R_1}$$ ，且总电阻 $$\frac{1}{R} = \frac{1}{R_1} + \frac{1}{R_2} \Rightarrow R = \frac{R_1R_2}{R_1+R_2}$$。
    * 并联中两分电阻阻值相等时总电阻有最大值 $$R_{\text{max}} = \frac{R_1}{2} = \frac{R_2}{2}$$。并联功率分配：$$\frac{P_1}{P_2} = \frac{R_2}{R_1}$$。
6. 电表改装定量计算：
    * **电压表量程扩大（串联分压电阻 $$R$$）**：
      $$R = (n - 1)R_g$$ ，其中扩大量程倍数 $$n = \frac{U}{U_g}$$。
    * **电流表量程扩大（并联分流电阻 $$R$$）**：
      $$R = \frac{R_g}{n - 1}$$ ，其中扩大量程倍数 $$n = \frac{I}{I_g}$$。

### （二）高阶电路测量实验深度复原
#### 1. 伏安法 (Voltmeter-Ammeter Method)
* **内接法**：电压表测的是 $$R_x + R_A$$ 的总电压， $$R_{\text{测}} = \frac{U}{I} = R_x + R_A$$。测量值偏大，适用于外接法误差大的**大电阻**（$$R_x \gg R_A$$）。
* **外接法**：电流表测的是 $$I_x + I_V$$ 的总电流， $$R_{\text{测}} = \frac{U}{I}$$。测量值偏小，适用于**小电阻**（$$R_x \ll R_V$$）。

#### 2. 伏阻法 (Double Voltmeter Method)
* **电路结构**：待测电阻 $$R_x$$ 与已知定值电阻 $$R_0$$ 串联。
* **实验操作与定量推导**：
    1. 当单刀双掷开关 $$S$$ 断开时，用电压表测得 $$R_x$$ 两端电压为 $$U_x$$。
    2. 当开关 $$S$$ 闭合时，将 $$R_0$$ 短路，此时测得总电压（即电源电压）为 $$U$$。
    3. 由串联分压关系知，定值电阻 $$R_0$$ 两端分得的电压为 $$U - U_x$$。
    4. 由于串联电路中电流处处相等，满足比例关系：$$\frac{U_x}{R_x} = \frac{U - U_x}{R_0}$$。
    5. 最终对待测电阻的定量求解计算式为：
       $$R_x = \frac{R_0 U_x}{U - U_x}$$

#### 3. 安阻法 (Double Ammeter Method)
* **电路结构**：待测电阻 $$R_x$$ 与已知定值电阻 $$R_0$$ 并联。
* **实验操作与定量推导**：
    1. 当开关 $$S$$ 与待测电阻支路接通断开时，通过电流表分别测出支路电流 $$I_x$$。
    2. 当开关 $$S$$ 完全闭合通电时，测得并联干路总电流为 $$I$$。
    3. 此时通过定值电阻 $$R_0$$ 支路的电流大小为 $$I - I_x$$（或者利用双电流表法分别记为 $$I_x$$ 与 $$I_0$$）。
    4. 由于并联电路各支路两端电压严格相等，满足方程：$$I_x R_x = (I - I_x) R_0$$ （或 $$I_x R_x = I_0 R_0$$）。
    5. 最终对待测电阻的定量求解计算式为：
       $$R_x = \frac{R_0 (I - I_x)}{I_x} \quad \text{或} \quad R_x = \frac{I_0}{I_x}R_0$$

#### 4. 替代法 (Substitution Method)
* **操作流程**：将待测电阻 $$R_x$$ 接入电路，闭合开关，记录此时电流表或电压表的示数为稳定值；然后断开开关，将 $$R_x$$ 替换为可调可读数的精密标准电阻箱 $$R_0$$，闭合开关后调节电阻箱的阻值，直到电流表或电压表的示数再次完全复原到先前的稳定值。此时，待测电阻的阻值在数值上完全等于电阻箱的读取阻值：
  $$R_x = R_0$$

#### 5. 电桥法 (Bridge Method)
* **操作流程**：构建惠斯通电桥平衡电路。调节桥臂已知电阻，当中间支路的精密检流计 $$\text{G}$$ 示数精准为零（即达到电桥平衡状态，中间支路无电流通过）时，对应的两对对角桥臂电阻乘积相等，即满足比例关系：$$\frac{R_1}{R_2} = \frac{R_3}{R_x}$$。
* **最终求解式**：
  $$R_x = \frac{R_2 R_3}{R_1}$$

#### 6. 欧姆表法 (Ohmmeter Method)
* **内部电路定量推导**：
    * 满偏调零时（指针指向最右侧 $$0\ \Omega$$ 刻度，此时两表笔直接短接 $$R_x = 0$$）：$$I_g = \frac{E}{R_g + r + R}$$。
    * 接入待测电阻 $$R_x$$ 测量时，电路总电流变为 $$I$$：$$I = \frac{E}{R_g + r + R + R_x}$$。利用电流 $$I$$ 与被测电阻 $$R_x$$ 的非线性反比对应关系，在表盘上直接标定阻值刻度。

### （三）电学保护电路定量计算案例
* **案例背景数据**：
  已知存在两个滑动变阻器（或用电器电阻原件）铭牌参数分别为：
  原件甲：“$$10\Omega\ 1\text{A}$$” ； 原件乙：“$$15\Omega\ 0.8\text{A}$$”。
* **串联限流安全电压与电流定量计算**：
    * 串联时，为了保证两元件均不被烧坏，整个串联电路所允许通过的最大安全电流不能超过其中较小的那一个额定电流：
      $$I_{\text{max}} = 0.8\text{ A}$$
    * 此时，整个电路两端允许加的最大安全总电压 $$U_{\text{max}}$$ 为：
      $$U_{\text{max}} = I_{\text{max}} \cdot (R_1 + R_2) = 0.8\text{ A} \times (10\Omega + 15\Omega) = 20\text{ V}$$
* **并联分流安全电压与电流定量计算**：
    * 并联时，两端所加的最大电压不能超过各原件中较小的那个额定电压值。
      原件甲的最大承受电压 $$U_1 = 1\text{ A} \times 10\Omega = 10\text{ V}$$ ；原件乙的最大承受电压 $$U_2 = 0.8\text{ A} \times 15\Omega = 12\text{ V}$$。
    * 故并联时允许加的最大安全电压为：
      $$U_{\text{max}} = 10\text{ V}$$
    * 此时并联电路对应的干路最大电流 $$I_{\text{max}}$$ 求解：
      通过甲的电流为 $$1\text{ A}$$，通过乙的电流为 $$\frac{10\text{ V}}{15\Omega} = \frac{2}{3}\text{ A}$$。
      故干路最大安全总电流为：
      $$I_{\text{max}} = 1\text{ A} + \frac{10}{15}\text{ A} \approx 1.67\text{ A}$$
    * 此时对应的电路最小总电阻为：
      $$R_{\text{min}} = \frac{U_{\text{max}}}{I_{\text{max}}} = \frac{10\text{ V}}{1.67\text{ A}} = 6\Omega \quad \left(\text{或由 } \frac{10 \times 15}{10 + 15} = 6\Omega \text{ 求得}\right)$$

### （四）家庭电路、安全用电与电能表
1. **家庭电路结构**：火线电压 $$220\text{ V}$$，零线电压 $$0\text{ V}$$。用电器采用并联。
2. 三孔插座接线：上地左零右火。三孔插座的接地线能将用电器外壳漏电电荷导入大地。
3. **熔断器（保险丝）**：接在火线上。特点为熔点低、电阻大。根据焦耳定律 $$Q=I^2Rt$$，当电路总电流过大时产生大量热，温度达到熔点自行熔断以保护电路。
4. **触电伤害类型**：
    * 低压触电：单线触电（火线 $$\rightarrow$$ 人体 $$\rightarrow$$ 大地）；双线触电（火线 $$\rightarrow$$ 人体 $$\rightarrow$$ 零线）。
    * 高压触电：高压电弧触电；跨步电压触电（高压线掉在地上落点周围形成电压差，人双脚跨步因电势差产生电流触电）。
5. **电能表（电度表）构造与工作机制**：
    * 结构包含：电流线圈（与负载串联）、电压线圈（与负载并联）、铝盘、永久磁铁、计数机构。
    * *机制*：接入交变电路时，电流与电压线圈在铁芯中产生交变磁场。交变磁场穿过铝盘感应出涡流，涡流受到磁场力的作用（安培力 $$F=BIL$$）得到转动力矩而驱动铝盘转动。铝盘转速与功率成正比。永久磁铁用于对铝盘转动产生电磁阻尼稳定转速。
    * 铭牌参数含义：“$$220\text{ V}\ 5(10)\text{A}\ 50\text{Hz}\ 1200\text{r}/(\text{kW}\cdot\text{h})$$” 表示额定电压 $$220\text{ V}$$，长时间通过的最大电流 $$5\text{ A}$$，短时间允许通过的最大电流 $$10\text{ A}$$，交流电频率 $$50\text{ Hz}$$，每消耗一整度电铝盘转过 1200 转。
