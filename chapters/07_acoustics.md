## 七、 声学 (Acoustics)

1. **声音的产生与传播**：
    * 一切正在发声的物体都在**振动**，振动停止，发声停止。
    * 声音作为**纵波**，其传播**需要介质**（以声波形式传播），**真空不能传声**。
    * *声速大小对照*：在 $$15^\circ\text{C}$$ 空气中 $$v_{\text{气}} = 340\text{ m/s}$$，水（液）中 $$v_{\text{液}} = 1500\text{ m/s}$$，钢铁（固）中 $$v_{\text{固}} = 3200\text{ m/s}$$。通常情况下 $$v_{\text{固}} > v_{\text{液}} > v_{\text{气}}$$。
    * 温度越高声音在空气中传播越慢。声波在界面会发生反射。人耳区分回声与原声的时间间隔必须在 $$0.15\text{s}$$ 以上。
2. **声音分类与特征**：
    * **乐音**：发声体做规则振动发出的声音。
        * **音调**：声音的高低，由发声体振动的**频率**（1秒内振动的次数，单位：赫兹，Hz）决定。
        * **响度**：声音的大小（单位：分贝，dB），由发声体的**振幅**以及距离发声体的远近决定。
    * **噪声**：发声体做无规则振动发出的声音。减弱噪声的三个途径：在声源处减弱、在传播途中减弱、在人耳处减弱。
3. **超声波与次声波**：人耳听觉范围为 $$20\text{ Hz} \sim 20,000\text{ Hz}$$。低于 $$20\text{ Hz}$$ 为次声波，高于 $$20,000\text{ Hz}$$ 为超声波。
4. 声波能传递**信息**与**能量**。

### （三）质点振动图像与机械波波动图像大比拼
| 特征维度 | 振动图像 (Vibration Graph) | 波动图像 (Wave Graph) |
| :--- | :--- | :--- |
| **研究对象** | 单个孤立的振动质点 | 介质中大量连续形成的传播质点群 |
| **物理内容** | 单个质点在运动过程中位移随时间的变化 | 某一特定时刻介质中各质点在空间坐标上的横向位移 |
| **横轴物理量**| 时间 ($$t$$) | 距离 ($$x$$) |
| **图线形态** | 随时间推移波形不断向右延伸延伸 | 经过一定时间后整体波形图沿传播方向平移（虚线法分析） |
| **物理意义** | 反映单个质点在各个不同时刻的运动位置 | 反映某一瞬间介质中空间各点的波形几何分布 |
