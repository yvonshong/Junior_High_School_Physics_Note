## 四、 机械能、动量与功 (Energy & Momentum)

### （一）功与功率
1. **做功的两个必要因素**：作用在物体上的**力**；物体在力的方向上通过的**一段距离**。
2. **功的大小计算**：通用公式 $$W = Fs\cos\theta$$ 或 $$W = Pt$$。单位为焦耳（$$\text{J}$$），$$1\text{ J} = 1\text{ N}\cdot\text{m}$$。
    * *功的原理*：使用任何机械都不省功（人们所做的功不少于不用机械直接手动做的功）。
3. **功率 $$P$$**：表示物体做功快慢的物理量。
    * 公式：$$P = \frac{W}{t} = \frac{Fs}{t} = Fv$$。单位为瓦特（$$\text{W}$$），$$1\text{ W} = 1\text{ J/s}$$。

### （二）机械效率与机械能
1. **功的分类**：有用功（$$W_{\text{有}}$$）为达到工作目的必须要做的功；额外功（$$W_{\text{额}}$$）为克服机械自重与摩擦不得不做但无用的功。总功 $$W_{\text{总}} = W_{\text{有}} + W_{\text{额}}$$。
2. **机械效率 $$\eta$$**：有用功跟总功的比值。
    * 公式：$$\eta = \frac{W_{\text{有}}}{W_{\text{总}}} \times 100\%$$。
    * *特性*：由于摩擦阻力存在，机械效率总是小于 $$100\%$$（符合热力学第三定律）。机械效率不是机械的固有属性，与被提升物重有关，与速度 $$v$$、高度 $$h$$ 无关。
3. **能量形式**：
    * 重力势能：$$E_p = Gh = mgh$$。
    * 弹性势能：$$E_p = \frac{1}{2}kx^2$$。
    * 动能：$$E_k = \frac{1}{2}mv^2$$。
    * **机械能守恒定律**：在没有摩擦阻力、只有重力或系统内弹力做功的条件下，系统机械能总量保持不变。
      公式：$$E_k + E_p = E_k' + E_p'$$ ；从高处自由下落时满足 $$mgH = \frac{1}{2}mv^2 \Rightarrow v = \sqrt{2gH}$$。

### （三）动能定理与动量守恒
1. **动能定理**：合外力做的功等于动能的变化量。$$\Sigma W = \Delta E_k = \frac{1}{2}mv_2^2 - \frac{1}{2}mv_1^2$$。合外力作正功，物体动能增加；合外力作负功，物体动能减小。
2. **重力做功与重力势能**：重力作正功，重力势能减小；重力作负功，重力势能增加（$$W_G = -\Delta E_p = mgh$$ 在数值上相等，功只与初末位置有关，与路径无关）。
3. **动量定理与守恒定律**：
    * 动量定理关系式：$$I = \Delta p \Rightarrow F\cdot t = mv - mv_0$$。动量（$$p=mv$$）与动能关系：$$p^2 = m^2v^2 = 2mE_k$$。
    * 动量守恒定律：系统不受外力或外力远小于内力时总动量守恒。$$m_1v_1 + m_2v_2 = m_1v_1' + m_2v_2'$$。若在某个特定方向上外力分量为零，则该方向上动量分量守恒。公式中速度必须是对地速度。
4. **碰撞精准定量解分类**：
    * **弹性碰撞**（无动能损失）：动量守恒且机械能守恒。
      若 $$m_2$$ 初状态静止（$$v_2 = 0$$），碰撞后的速度精准解为：
      $$v_1' = \frac{(m_1 - m_2)v_1}{m_1 + m_2}$$
      $$v_2' = \frac{2m_1v_1}{m_1 + m_2}$$
        * *特例*：若 $$m_1 = m_2$$，则 $$v_1' = 0, v_2' = v_1$$（两球完全交换速度）。
        * *特例*：若 $$m_1 > m_2$$，则 $$v_1' > 0, v_2' > 0$$（两球均向前运动）。
        * *特例*：若 $$m_1 < m_2$$，则 $$v_1' < 0, v_2' > 0$$（质量小的 $$m_1$$ 被反弹回来）。
        * *特例*：若 $$m_2 \rightarrow \infty$$，则 $$m_1$$ 以原速率反向弹回。
    * **完全非非弹性碰撞**：碰后两物体粘在一起以相同速度运动（$$m_1v_1 + m_2v_2 = (m_1 + m_2)v'$$），这种情况系统总动能损失最大，损失的动能完全转化为内能。
    * **一般碰撞**：仅满足动量守恒 $$m_1v_1 + m_2v_2 = m_1v_1' + m_2v_2'$$。
