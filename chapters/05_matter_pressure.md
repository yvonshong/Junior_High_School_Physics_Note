## 五、 质量、密度与流体压强 (Matter & Pressure)

### （一）质量与密度
1. **质量 $m$**：物体所含物质的多少。质量是物体的基本属性，不随物体的形状、状态、位置而改变。测量工具为天平（等臂杠杆）。
    * *天平调平与使用规范*：底座水平、游码拨零、横梁调平（左偏右调）。左物右码。加减砝码从大到小，最后移动游标使横梁平衡。求和时物重等于砝码总重加游标对应的阻值。若物码放反，阻值满足 $m_{\text{物}} = m_{\text{砝码}} - m_{\text{游标}}$。
2. **体积 $V$**：物体对空间占有的大小。单位为立方米。
3. **密度 $\rho$**：单位体积某种物质的质量，可用于区分物质种类。
    * 计算公式：$\rho = \frac{m}{V}$。单位换算：$1\text{ g/cm}^3 = 10^3\text{ kg/m}^3$。
    * 混合物密度：$\rho_{\text{混}} = \frac{m_{\text{总}}}{V_{\text{总}}} = \frac{m_1 + m_2 + \cdots + m_n}{V_1 + V_2 + \cdots + V_n}$。
    * 密度取决于物质种类和状态，与质量 $m$ 和体积 $V$ 无关。可以通过比较实际体积与纯物质体积来解决空心问题。

### （二）固体压强与液体压强
1. **压力 $F$**：垂直作用在物体表面上的力，方向指向被压物体。
    * 大小分析：水平面上受重力静止的物体压力 $F = G$；受到竖直向上拉力 $F_1$ 时的压力 $F = G - F_1$；在斜面上静止时的压力 $F = G\cos\theta$。
2. **压强 $P$**：单位面积上受到的压力大小，表示压力的作用效果。
    * 通用公式：$P = \frac{F}{S}$。对于水平面上形状规则、密度均匀的柱体：$P = \frac{mg}{S} = \frac{\rho Vg}{S} = \frac{\rho Shg}{S} = \rho gh$。
    * 单位：帕斯卡（$\text{Pa}$），$1\text{ Pa} = 1\text{ N/m}^2$。
    * *固体与液体的压强压力传递特性*：固体上传递压力，密闭液体上传递压强。
3. **液体压强**：
    * *产生原因*：由于液体受到重力作用且具有流动性，液体对容器底和侧壁、向各个方向都有压强。
    * *大小公式*：$P = \rho gh$，深度相同时液体各方向压强相等。
    * **连通器**：上端开口、底部连通的容器。内装同种液体不流动时，液面总相平。
    * **帕斯卡原理**：加在密闭液体上的压强，能大小不变地由液体向各个方向传递（$P_1 = P_2 \Rightarrow \frac{F_1}{S_1} = \frac{F_2}{S_2} \Rightarrow \frac{F_1}{F_2} = \frac{S_1}{S_2}$）。

### （三）大气压强与经典气体实验定律
1. **大气压强**：
    * *产生原因*：空气受到重力作用且具有流动性，分子热运动撞击。
    * 马德堡半球实验证明了大气压强的存在且巨大。
    * 托里拆利实验测定其大小：标准大气压 $\text{1 atm} = 101,325\text{ Pa} \approx 10^5\text{ Pa} = 76\text{cmHg}$。实验结果与管子粗细、形状、倾斜与否、上提下压均无关；若管内漏气则测量值偏小。
    * 大气压随高度增加而减小。液体的沸点随气压增大而升高，随气压减小而降低。
    * **流体压强与流速关系**：流体中流速大的地方压强小，流速小的地方压强大。
    * 对于一定质量的气体，压强与体积成反比：$P \propto \frac{1}{V}$。
2. **气体实验三大定律深度对照大表**：

| 定律名称 | 实验控制条件 | 物理规律描述 | 数学表达式 |
| :--- | :--- | :--- | :--- |
| **玻意耳定律** | 温度 $T$ 恒定，一定质量气体 | 体积与压强成反比 | $P_1V_1 = P_2V_2$ 或 $\frac{P_1}{P_2} = \frac{V_2}{V_1}$ |
| **查理定律** | 体积 $V$ 恒定，一定质量气体 | 压强与热力学温度成正比 | $\frac{P_1}{P_2} = \frac{T_1}{T_2}$ 或 $P_t = P_0\left(1 + \frac{t}{273}\right)$ |
| **盖-吕萨克定律** | 压强 $P$ 恒定，一定质量气体 | 体积与热力学温度成正比 | $\frac{V_1}{V_2} = \frac{T_1}{T_2}$ 或 $V_t = V_0\left(1 + \frac{t}{273}\right)$ |

* **理想气体状态方程通用式**：$\frac{P_1V_1}{T_1} = \frac{P_2V_2}{T_2} = k$。
* **理想气体状态方程推论**：$\frac{P_1}{\rho_1 T_1} = \frac{P_2}{\rho_2 T_2}$ （其中 $\rho$ 为气体质量密度）。
