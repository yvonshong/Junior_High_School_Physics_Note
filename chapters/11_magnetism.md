## 十一、 磁学与电磁感应 (Magnetism)

### （一）磁场与安培力
1. 磁体吸引铁钴镍等物质。磁场强弱由磁感应强度 $$B$$ 表示（单位：特斯拉，T）。$$B$$ 为矢量，方向规定为放在该处的磁针北极（N极）的受力指向。磁感线在空间永不相交，外磁路从 $$\text{N} \rightarrow \text{S}$$，内磁路从 $$\text{S} \rightarrow \text{N}$$。
2. **通电导线在磁场中受到的安培力**：
    * 当电流方向与磁场方向垂直时：$$F = BIL$$。
    * 当电流方向与磁场方向平行时：$$F = 0$$。
    * 当夹角为 $$\theta$$ 时（将 $$B$$ 分解为垂直于电流的分量）：$$F = BIL\sin\theta$$。
    * 方向由**左手定则**判定：展开左手让磁场方向垂直穿过手心，四指指向电流方向，大拇指所指的方向就是力的方向。
3. **电动机模型**：定子（磁铁）、转子（线圈）、电刷、换向器。
    * 当线圈平面与磁场垂直时：力矩 $$M = 0$$。
    * 当线圈平面与磁场平行时：最大转动力矩 $$M = BIS$$。

### （二）洛伦兹力与电磁感应定量公式
1. **洛伦兹力**：磁场对运动电荷的作用力。大小 $$F = qvB$$（垂直时），方向由左手定则判定（正电荷运动方向为电流方向，负电荷相反）。**洛伦兹力只改变速度方向、永远不做功，物体动能保持不变**。
    * 匀强磁场中做圆周运动的轨道半径：$$r = \frac{mv}{qB}$$。
    * 运动周期：$$T = \frac{2\pi m}{qB}$$。
2. **法拉第电磁感应感应电动势 $$E$$ 的定量求法**：
    * **动生切割**：$$E = BLv\sin\theta$$。方向由**右手定则**判定：展开右手让磁场垂直穿过手心，大拇指指向导线运动方向，四指伸展方向即为感应电流方向。
    * *特殊切割*（直导线绕一端匀速转动切割）：$$E = \frac{1}{2}BL^2\omega$$。
    * **感生磁通量变化（法拉第定律）**：$$E = \frac{\Delta \Phi}{\Delta t} = B\cdot\frac{\Delta S}{\Delta t} = S\cdot\frac{\Delta B}{\Delta t}$$。
    * **自感电动势**：$$E_{\text{自}} = L\frac{\Delta I}{\Delta t}$$。
3. **正弦交流电与理想变压器**：
    * 交流电最大值：$$E_{\text{max}} = NBS\omega$$。瞬时值：$$e = E_{\text{max}}\sin\omega t$$ ， $$i = I_{\text{max}}\sin\omega t$$。
    * 有效值换算：$$E = \frac{E_{\text{max}}}{\sqrt{2}}$$ ， $$U = \frac{U_{\text{max}}}{\sqrt{2}}$$ ， $$I = \frac{I_{\text{max}}}{\sqrt{2}}$$。
    * **理想变压器定量方程**：
        * 电压比：$$\frac{U_1}{U_2} = \frac{n_1}{n_2}$$
        * 功率守恒：$$P_{\text{入}} = P_{\text{出}} \Rightarrow U_1I_1 = U_2I_2$$
        * 单个副线圈电流比：$$\frac{I_1}{I_2} = \frac{n_2}{n_1}$$ ；若有多个副线圈时电流满足 $$P_1 = P_2 + P_3 + \cdots$$。
4. **电磁振荡与电磁波**：
    * $$\text{LC}$$ 回路振荡周期与频率：$$T = 2\pi\sqrt{LC}$$ ， $$f = \frac{1}{2\pi\sqrt{LC}}$$。
    * 电磁波波长与光速关系：$$\lambda = \frac{C}{f} = CT$$。
